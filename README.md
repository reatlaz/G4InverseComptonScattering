## Installation instructions:

This guide will assume the unpacked *Geant4* source package is called "geant4-v11.0.X"
1. Add the .cc files to `geant4-v11.0.X/source/processes/electromagnetic/xrays/src`
2. Add the .hh files to `geant4-v11.0.X/source/processes/electromagnetic/xrays/include`
3. Install *Geant4* from source
